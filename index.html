LENGUAJES DE PROGRAMACIÓN


+

HTML 5


Este lenguaje de programación me permite maquetar un sitio web, sus siglas (HTML) significan 'HyperText Markup Language', en español 'Lenguaje de Marcado e HiperTexto'.

+

CSS

CSS es un lenguaje de estilo. Este lenguaje le estilos al cuerpo de nuestro sitio web. Es el maquillaje de HTML. Todo el diseño de un sitio va directamente relacionado con este lenguaje. Sus siglas (CSS) también tienen un significado 'Cascading StyleSheet', su traducción es 'Hojas de estilo de cascada'.

+

JavaScript

Javascript es una especie de lenguaje de programación ligera, interpretado por la mayoría de los navegadores y que les proporciona a las páginas web, efectos y funciones complementarias a las consideradas como estándar HTML.

+

Bootstrap

Bootstrap es un framework CSS y Javascript diseñado para la creación de interfaces limpias y con un diseño responsive. Además, ofrece un amplio abanico de herramientas y funciones, de manera que los usuarios pueden crear prácticamente cualquier tipo de sitio web haciendo uso de los mismos.

+

WordPress


Este aplicativo me permite crear un blog, webs empresariales, tiendas online, periódico digital, central de reservas, etc.
